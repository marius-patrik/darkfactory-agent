# AI-Assisted Development Plan

This project uses AI-assisted code generation as permitted by the OpenDAW upstream project.

All significant AI-assisted contributions are documented in this `/plans` directory.

## Process

1. Specs are written and locked before implementation.
2. Implementation is decomposed into focused subtasks.
3. Generated code is reviewed for correctness, license compliance, and alignment with specs.
4. Tests verify behavior before merging.

## Contributors

All human contributors must understand every line of code they submit, in accordance with OpenDAW's contribution guidelines.
